# Where is my pointer
This script encircles the pointer with a large yellow square that shrinks towards the pointer.

I script ought to be runed with the ```timeout``` command.

## Build
Requires libX11
```sh
gcc -std=c99 WhereIsMyPointer.c -o wimpointer -lX11
```
